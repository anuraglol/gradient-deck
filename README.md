<h2>Cloning/Installation Steps:</h2>

<h3>Clone the repo</h3>

```
git clone https://github.com/kr-anurag/gradient-deck
```

<h3>Change the directory</h3>

```
cd gradient-deck
```

<h3>Install the required modules</h3>

```
npm install
```

<h3>Run the app</h3>

```
npm run dev

// or if you use yarn
yarn dev
```
